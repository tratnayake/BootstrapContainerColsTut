What does the col-small-4 mean?






The XS,SMALL,LARGE indicates what screen size this column size applies to.

e.g. "col-sm-4" means that this elment will span 4 columns on a SMALL screen size.

According to boostrap:

col-xs

Phones

Any device with a width < 768px

col-sm

Tablets

Any device with a width > 768px

col-md

Laptop/Desktop

Any device with a width > 990px

col-lg

Big Laptop/Desktop

Any device with a width > 1200px
